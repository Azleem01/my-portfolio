BIO

Grounded AI work that earns trust.

I build tools and prototypes that are defined by clarity, not cleverness. That means clear assumptions, robust handoffs, and honest communication on what the model can — and cannot — do.

As technical lead on Sentinel AI, I delivered an emergency-department triage prototype for the Imperial College Global Challenge Lab. I owned synthetic-data model training, a FastAPI scoring service, and a React triage board designed to help clinicians make faster, safer decisions.

I also build production-facing AI products like Flashcard AI, where screenshot-to-flashcard workflows must run reliably for students, and research tools like Anti-Deepfake Protector, which makes image editing harder for malicious models.
